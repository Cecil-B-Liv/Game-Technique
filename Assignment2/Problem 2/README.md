# Assignment 2 - Problem 2: Connect4 MCTS

This project implements a Monte Carlo Tree Search (MCTS) agent to play Connect4.

## Structure
- `main.py` - Entry point to run the game and test the MCTS agent.
- `game.py` - Contains the Connect4 game logic.
- `mcts.py` - Implements the MCTS algorithm.
- `config.py` - Configuration settings for the game and agent.
- `Labcode/connect4_mcts.py` - Additional lab code for Connect4 MCTS.

## How to Run
1. Make sure you have Python 3 installed.
2. Run the main script:
   ```bash
   python main.py
   ```

## Features
- Play Connect4 against an MCTS agent.
- Easily configurable parameters in `config.py`.

## Requirements
- Python 3.x
- No external dependencies required.
